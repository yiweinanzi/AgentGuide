# Agent Papers

This directory contains a curated list of academic papers related to AI Agents, categorized by research area.

## Categories

- [Agent RL](./agent_rl/README.md) - Papers focusing on Reinforcement Learning for agent decision-making.
- [Agent Memory](./agent_memory/README.md) - Research on memory mechanisms for long-term context and learning.
- [Data Synthesis](./data_synthesis/README.md) - Papers on methods for generating synthetic data for training agents.
